## Setup
       cd
       git clone https://github.com/esmanchik/emacs.git
       ln -s emacs/.emacs
       cd .emacs.d
       wget https://ergoemacs.googlecode.com/files/ergoemacs_1.9.3.1.zip
       unzip ergoemacs_1.9.3.1.zip
       ln -s ../emacs/vendor
